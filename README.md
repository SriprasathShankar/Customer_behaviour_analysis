4\. üìù Project Steps and Methodology

The project was executed in a clear, four-stage pipeline:
A. Data Loading & Cleaning (Python)

  * Loaded the raw CSV file into a Pandas DataFrame.
  * Performed Exploratory Data Analysis (EDA) to check distributions, identify outliers, and assess data quality.
  * Handled missing values (`NaN`) through imputation or removal.
  * Standardized text formats and corrected data types (e.g., converting strings to datetime objects).

B. Database Integration (MySQL)

  * Designed a normalized schema to separate data into multiple tables (e.g., `customers`, `orders`, `products`).
  * Used a Python script (`mysql.connector`) to efficiently **load the cleaned data** from the DataFrame into the prepared MySQL tables.
  * Executed complex SQL queries to join tables, calculate key metrics, and prepare aggregated data for visualization.

C. Metric Calculation & Analysis

Key analysis and metrics calculated include:

  * Customer Revenue: Total revenue generated by individual customers.
  * Order Frequency: Average number of orders per month.
  * Product Performance: Top-selling products by volume and revenue.
  * Geographic Analysis: Revenue split by region.

D. Power BI Dashboard Development

  * Connected Power BI directly to the MySQL database tables.
  * Built a robust Data Model with appropriate relationships between the `customers`, `orders`, and `products` tables.
  * Created calculated measures (DAX) for dynamic reporting (e.g., Year-over-Year Growth).
